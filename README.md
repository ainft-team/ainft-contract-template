# ainft-contract-template

### Install dependencies

```sh
$ yarn
```

### Compile

```sh
$ yarn compile
```

### Lint Solidity

```sh
$ yarn lint:sol
```

### Apply prettier

```sh
$ yarn prettier
```

### Run tests

```sh
$ yarn test
```

### Get a test coverage

```sh
$ yarn coverage
```

### Delete smart contract artifacts, coverage reports, and hardhat cache

```sh
$ yarn clean
```
